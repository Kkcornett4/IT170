# IT170
Final project for IT 170

In this project we will be creating a system that takes input as an image or a folder of files. There will be two separte options for the code including taking the text from an image to a file or organizing the folder of files by type and date. The goal of this project is to help students and teachers store their needed information in a way that is easy to find aspects. If time allows there will be an additional aspect using AI to summarize files and key word extaction.
